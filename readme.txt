# 📊 Fundamental Screener

米国株のファンダメンタル分析ツール。Alpha Vantage（無料API）からリアルデータを取得し、5カテゴリでスコアリング・比較します。

## 機能

- **Alpha Vantage実データ取得** — OVERVIEW / 損益計算書 / 貸借対照表 / CF計算書
- **5カテゴリスコアリング（0〜10点）**
  - 収益性（ROE、営業利益率、FCFマージン）
  - 成長性（売上3年CAGR、純利益成長、FCF成長率）
  - 財務健全性（流動比率、D/E比率、Piotroski Fスコア）
  - バリュエーション（P/E、PEG、P/B、EV/EBITDA）
  - マクロ適合性
- **比較テーブルビュー** — 複数銘柄を横並びで一覧比較、最良値に▲マーク
- **カードビュー** — 銘柄ごとの詳細スコア表示
- **AI投資論文生成**（Anthropic APIキー入力時）— 1〜2年保有観点の日本語投資論文

## 使い方

### 1. リポジトリをクローン
```bash
git clone https://github.com/YOUR_USERNAME/fundamental-screener.git
cd fundamental-screener
```

### 2. ブラウザで開く
```bash
open index.html
# または VS Code Live Server で起動
```

### 3. APIキーを入力
- **Alpha Vantage APIキー**（必須）: [無料登録](https://www.alphavantage.co/support/#api-key)
- **Anthropic APIキー**（任意）: AI投資論文生成に使用

### 4. ティッカーを入力して分析開始
```
例: AAPL MSFT NVDA GOOG AMZN
```

## API制限（無料枠）

| 項目 | 内容 |
|---|---|
| リクエスト数 | 25 req/日、5 req/分 |
| 1銘柄あたり | 4リクエスト消費 |
| 複数銘柄 | 自動で15秒インターバル |

## スコアリング基準

### 収益性（×25%）
| 指標 | 10点 | 7点 | 5点 | 2点 |
|---|---|---|---|---|
| ROE | >25% | >18% | >12% | <7% |
| 営業利益率 | >25% | >15% | >8% | <3% |
| FCFマージン | >20% | >12% | >5% | <0% |

### 成長性（×30%）
| 指標 | 10点 | 7点 | 5点 | 2点 |
|---|---|---|---|---|
| 売上CAGR(3Y) | >25% | >15% | >8% | <3% |
| 純利益成長 | >25% | >15% | >8% | <0% |
| FCF成長率 | >20% | >10% | >3% | <0% |

### 財務健全性（×20%）
| 指標 | 10点 | 7点 | 5点 | 2点 |
|---|---|---|---|---|
| 流動比率 | >2.5 | >2.0 | >1.5 | <1.0 |
| D/E比率 | <0.3 | <0.7 | <1.0 | >1.5 |
| Piotroski F | 7/7 | 5-6/7 | 3-4/7 | <3/7 |
